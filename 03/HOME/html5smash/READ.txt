В файле index.html указана структура  HTML-документа


Задание
1. Используя семантические теги
2. Дать html элементам названия классов. Старайтесь давать
   html-єлементам такие имена, чтобы по этим именам любой разработчик, который будет работать с Вашим кодом понимал, какой контент находится в этом элементе

   Например
     <article>
        <h2 class="featured-article-title">Featured Article</h2>
        ...


3. Текст, обрамленный в __  __   - это гиперссылка.
   Например __HTML5__  Вы можете оформлять как
         <a href="#">HTML5</a>
   Постарайтесь найти через google поиск реальные URL на тот контент, который в ссылках
   Например HTML5 => это
      <a href="https://www.w3.org/TR/html5/">HTML5</a>


4. В каталоге css  создать файл style.css и подключить его в
   index.html



======== Особенности  !!!
1. Разделы сайта  указаны цифрами 1,2,3 ...
   (впоминаем какие  элементы создают разделы HTML документа)

2. Не забываем про такой атрибут для ссылок как target, если
   ссылка ведет на внешний ресурс

3. Если у статьи (впрочем как ми у любой другой публикации есть
   дата) то для нее используется тег <time></time> c атрибутом
    datetime="YYYY-MM-DDTHH:MM:SS"  где T - разделитель даты и времени

  Например  дата 12 апреля 2017 года 18 часов 22 минуты 55 секунд

       <time datetime="2017-04-12T18:22:55">12 апреля 2017</time>

   А если время не имеет значения, то
      <time datetime="2017-04-12">12 апреля 2017</time>

4. Не забываем про тег <main></main> в который мы включаем основное  содержимое сайта.

5. Не забываем, что если контент является перечислением чего бы то ни было (например названия популярных постов в блоге, список ссылок) - то желательно использовать списки <ul> или <ol>


6. Результат работы проверить с помощью HTML 5 Outliner
   https://gsnedders.html5.org/outliner/
